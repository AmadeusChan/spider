骨头出事不必急——
骨头也能再造了
科技日报 2001年10月11日
骨骼损伤、缺损是骨科常见的临床问题。用于骨缺损修复的植骨术，在美国已成为仅次于输血的组织移植技术。由于自体植骨来源极为有限，又增加创伤，因而很多国家都采用保存的同种异体骨移植术，但也存在来源困难、有免疫排斥反应、传播疾病等问题。我国科学家采用人工合成的载体框架、生长因子和患者自己的细胞制造活的人工骨使这个问题得到了很好的解决。 
骨骼损伤、缺损是骨科每天都要面临的临床问题。用于骨缺损修复的植骨术，在美国已成为仅次于输血的组织移植技术。由于自体植骨来源极为有限，又增加创伤，因而很多国家都采用保存的同种异体骨移植术，但也存在来源困难、有免疫排斥反应、传播疾病等问题。骨组织工程方面的研究成果通过采用人工合成的载体框架、生长因子和患者自己的细胞制造活的人工骨使这个问题得到了很好的解决。由清华大学机械工程系颜永年教授负责的“组织工程材料的大段骨快速成形制造”项目是由清华大学机械工程系激光快速成形中心牵头，由第四军医大学全军骨科研究所和清华大学材料科学与工程系参加的一个医工结合的交叉学科研究项目，经过两年多的努力，取得了突破性的创新成果。他们通过新的快速成形工艺开发了与人骨组织的孔隙结构和材料结构高度类似的大段人工骨修复物，完成了兔桡骨15mm节段性缺损修复实验和犬桡骨20mm节段性缺损修复实验，正在积极准备进入临床实验研究。 
颜永年，男，1938年1月12日生于重庆市。1961年毕业于清华大学机械系。1987－1988年为美国加州大学洛山矶分校（ U CLA）访问学者。清华大学机械系材料加工工程及自动化研究所所长，教授，博士生导师。提出国际广泛应用的超高压容器的等剪应力缠绕理论和计算方法，完成5千至4万吨压机20余台，获部级、国家级奖。担任863计划   C IMS主题广东华宝空调器厂物流子系统负责人，获省级一等奖和国家级二等奖。国内外知名快速成形技术专家，完成 S SM等4项快速成形工艺和多种型号设备的研发并实现产业化，多功能快速成形、双激光并行扫描和无木模铸型制造等工艺及设备为世界首创。提出生物制造工程概念和技术体系，承担国家高技术计划项目
“组织工程材料的大段骨快速成形制造”，鉴定为国际领先。任中国机械工程学会电加工分会快速成形技术委员会主任。 “组织工程材料的大段骨快速成形制造”项目内容：采用与天然骨成份组织相同的人工骨组织工程材料，根据病人骨破损处的   C T数据重构骨结构的三维功能梯度模型，经数据处理，通过先进的数字喷射快速成形技术完成骨组织框架结构的成形制造和生长因子的复合过程，通过体外细胞培养、小动物和大动物植入实验，研究框架结构降解与成骨过程的基本科学问题，达到可进入临床实验水平。

人工器官———现代医学新领域 
在外科临床病例中，由于衰老、病变、创伤、肿瘤切除、骨髓炎所致的器官缺损十分常见，因而广泛需要对人体器官进行人工替代。这种替代可以分为自体或异体移植和采用人工器官两种途径。现代外科医学的历史发展是从破坏性、创伤性的切除手术，趋向于修复重建外科新领域。 
20世纪下半叶在修复和治疗人体各种组织、器官畸形和疾病方面有着很大的发展，其中一个重要手段就是组织和器官的移植。皮肤组织的移植开创于19世纪后期，盛行于20世纪。从最早的自体皮肤移植到异体或异种皮肤移植以治疗抢救大面积烧伤病人，得到广泛的应用，且至今受到重视。从皮肤移植开始，后来又扩展到人体各种组织的移植，如软骨、骨骼、筋膜、肌腱血管、神经和肌肉等。这些自体或异体组织移植，一直是各国外科医师修残补缺、恢复功能和形态的主要方法之一。 
虽然自体组织和异体器官移植在临床应用上已获得很大发展，但还存在一些明显的缺陷，就是自体组织必须以牺牲个人的正常组织为代价，异体组织或器官则又存在免疫排斥反应和供体器官严重不足的问题。 
人工器官在近三十年来，得到了广泛的应用。一个新兴的人工器官产业正在兴起，其市场销售额以每年15％的速率递增。1992年，世界生物材料和人工器官的销售额就达500亿美元。 
由于人工器官的临床应用，拯救了成千上万患者的生命，减轻了疾病给患者及其家属带来的痛苦与折磨，引起了医学界的广泛重视，加快了人工器官的研究步伐。 
生物医学工程在本世纪将成为继信息产业之后最重要的科学研究和经济增长的热点，其中生命体的人工合成和人体器官的人工替代成为了全球瞩目的学科前沿。

“仿生大段骨”成功研制 
我国科学家的“组织工程材料的大段骨快速成形制造”项目进行的是骨组织工程方面的研究。骨骼损伤、缺损是骨科每天都要面临的临床问题。用于骨缺损修复的植骨术，在美国已成为仅次于输血的组织移植技术。由于自体植骨来源极为有限，又增加创伤，因而很多国家都采用保存的同种异体骨移植术，但也存在来源困难、有免疫排斥反应、传播疾病等问题。骨组织工程方面的研究成果使这个问题得到了很好的解决。主要的研究目标是： 
1、开发并选择合适的骨组织工程材料，通过软件开发实现骨 C T图像的三维重构和骨微孔结构的设计； 
2、根据快速成形的工艺要求改进骨组织工程材料的成形性能，开发与材料相适应的新的快速成形工艺，成形长度不少于100mm的骨组织工程载体框架结构，使其在材料、微观结构等方面与人骨高度类似； 
3、采用上述新工艺制备的载体框架，通过与 B MP的多维复合，制备成具有良好的 B MP缓释能力、生物相容性、生物降解性能、骨传导性能和骨诱导性能仿生人工骨，完成兔桡骨15mm和犬桡骨20mm节段性缺损的修复实验，达到可以进入临床实验阶段的科学要求。 
“组织工程材料的大段骨快速成形制造”是采用与天然骨成份组织相同的人工骨组织工程材料，根据病人骨破损处的 C T数据重构骨结构的三维功能梯度模型，经数据处理，通过先进的数字喷射快速成形技术完成骨组织框架结构的成形制造和生长因子的复合过程，通过体外细胞培养、小动物和大动物植入实验，研究框架结构降解与成骨过程的基本科学问题，达到可进入临床实验水平。 
科学家们所取得的研究成果主要可以归纳为：骨 C T图像数据三维几何重构、胶原———羟基磷灰石纳米晶材料成形性的改造、成形设备和成形工艺、生长因子与成形材料的复合、动物实验及细胞培养实验共五个方面取得进展。自项目启动至2000年3月，清华大学激光快速成形中心的一个软件开发小组结合骨解剖学   C T图像的特点，开发了适合骨 C T图像数据的三维几何重构软件，并在孔隙梯度填充设计、预处理等五个方面具有特色。运用此软件，结合快速成形技术和临床手术，完成了全国首例仿生半骨盆置换。 
在这个项目的研究中开发了具有与人骨高度类似的纳米层状结构的纳米晶羟基磷灰石／胶原复合材料作为骨组织工程材料。为了将纳米晶羟基磷灰石／胶原复合材料制成可供骨细胞培养的框架结构，其可成形性和成形方法的研究是极其重要的步骤。 
为了将纳米晶羟基磷灰石／胶原复合材料根据实际骨 C T重构模型制成载体框架结构，在2000年3月至8月期间，清华大学激光快速成形中心开发了一种速成形新工艺：低温挤压／喷射堆积成形工艺，研制了适合此工艺的设备、喷头等及相应的控制软件。 
从2000年9月开始，通过快速成形制备大段骨载体框架结构，采用真空吸附法复合 B MP，制备了用于动物实验的仿生大段骨试样，分别进行了小动物和大动物实验，即修复兔桡骨15mm节段性缺损实验和修复犬桡骨20mm节段性缺损实验，均取得了令人满意的结果。
 2001年5月，颜永年教授访问了许多世界一流大学从事同类研究的实验室，这些实验室的研究还处于起步阶段，尚未进行动物实验。通过技术查新，同样证明这个项目研究处于国际领先水平。

人工器官前景广阔 
下个世纪，以人的健康为中心的产业将节节上升，其规模和产值将超过汽车行业，达到与信息产业和旅游业相媲美的程度。 
人工器官在医学上的广泛应用，向科学界提出了新的要求。制造出具有优异的组织相容性和生物活性，能诱导缺损组织或器官的再生成而自身却能被降解和吸收的人工器官再生载体，即我们所说的组织工程人工器官的细胞载体框架结构越来越受到学术界和工业界的重视。可以想象，一旦这种细胞载体框架结构制造出来，将能使许多不治之症的治愈成为可能，这本身也意味着对人工器官的需求更加扩大。 
我国是世界上人口最多的国家，对于人工器官的需求很高。对于大段骨这样的器官，如果能够采取本项目的技术路线制造出来，将满足极大的社会需求。 
虽然我国的人均汽车拥有量还较低，但是交通事故率却不低，加之疾患特别是老年性骨科病，可以肯定我国是大段人工骨的最大需求国。我们的大段骨主要能用于以下四方面疾病的治疗： 
一、骨肿瘤病人切除手术后的保肢治疗。现在大部分这类病例只能截肢，给病人今后的生活带来了极大的痛苦，而运用我们的组织工程材料大段骨能够成功的进行保肢治疗。 
二、人工关节的置换。目前所广泛使用的人工关节多为钛金属和陶瓷材料制造的人工关节，这些关节在生物相容性和机械性能方面还存在许多问题，一般使用寿命不超过10年；运用我们的技术制造的组织工程材料人工关节可以促进人体关节的再生，并不存在使用寿命的问题，必将替代现在使用的人工关节。目前，国内每年人工髋和人工膝关节的用量很大。由此我们不难看出我们的新型人工关节在投入市场后必将带来可观的经济效益。 
三、各种不能愈合的严重机械骨外伤的治疗。运用我们的组织工程材料大段骨技术能成功的进行指（趾）骨的再造、颅骨的修复和大段骨缺损的再造和修复。 
四、用于整形外科手术，如颌面的先天和后天缺损与畸形和牙龈的修复等等。随着人民生活水平的迅速提高，整形外科手术已越来越为广大患者所接受，可以预计这一方面的市场需求在未来很长时期会呈持续上升趋势。
 除了和平时期的疾病损伤对于组织工程材料大段人工骨有很大的需求之外，由于新型重型武器的不断推陈出新，战时的大段骨缺损也会越来越多。我们的组织工程材料大段人工骨技术的推广和应用无疑有利于我国国防医疗水平的提高，同时也会带来可观的经济效益。
